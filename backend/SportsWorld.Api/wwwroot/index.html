SportsWorld Web API

This API provides functionality for managing athletes, venues, finances and purchases in the SportsWorld application.

Base URL

http://localhost:5189

API Explorer

For interactive API testing and detailed documentation, visit the Swagger UI.

Endpoints

AthleteController

GET /api/Athlete

Get all athletes.

GET /api/Athlete/{id}

Get a specific athlete by ID.

GET /api/Athlete/search?name={name}

Search athletes by name.

POST /api/Athlete

Create a new athlete.

PUT /api/Athlete/{id}

Update an athlete or mark as purchased.

DELETE /api/Athlete/{id}

Delete an athlete.

POST /api/Athlete/{id}/purchase

Purchase an athlete.

VenueController

GET /api/Venue

Get all venues.

GET /api/Venue/{id}

Get a specific venue by ID.

GET /api/Venue/search?name={name}

Search venues by name.

POST /api/Venue

Create a new venue.

PUT /api/Venue/{id}

Update a venue.

DELETE /api/Venue/{id}

Delete a venue.

FinanceController

GET /api/Finance

Get the finance data.

GET /api/Finance/{id}

Get finance data by ID.

POST /api/Finance

Create finance data.

PUT /api/Finance/{id}

Update finance data.

DELETE /api/Finance/{id}

Delete finance data.

POST /api/Finance/loan

Take a loan.

UploadImageController

POST /api/UploadImage?folder={folder}

Upload an image to a specified folder (default: venues).

ResetController

POST /api/Reset

Reset finance and athlete purchase statuses.
